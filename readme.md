# django ecommerce
ecommerce web app using django mltiple sellers
## technologies-used
* backend
  * django
  * mysql
* frontend
  * HTML5
  * Css3
  * JavaScript
## features
* product
  * add product
  * delete product
  * update product
* review & rate product
* cart
  * handling cart features using django sessions
  * add to cart
  * remove from cart
  * clear cart
* accounts
  * sign up
  * log in
  * log out
  * reset password
  * delete account
